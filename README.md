# Hotel Ganges Grand Website

## 📌 Project Overview

Hotel Ganges Grand is a responsive hotel website developed to provide an online platform where users can explore hotel services, room details, amenities, restaurant facilities, gallery, and nearby attractions. The website also includes a booking inquiry system that allows customers to submit their details digitally.

The project focuses on creating a clean, responsive, and user-friendly interface while providing an efficient way to collect and manage customer inquiries.

## ✨ Key Features

* Fully responsive design for different screen sizes
* Interactive and user-friendly interface
* Dedicated sections for:

  * Hotel overview
  * Rooms and facilities
  * Restaurant services
  * Gallery
  * Amenities
  * Nearby attractions
  * Contact information
* Online booking inquiry form
* Automated storage of customer inquiries using Google Sheets integration

## 🛠️ Technologies Used

* **HTML5** – Used for structuring web pages and content
* **CSS3** – Used for styling, layouts, and responsive design
* **JavaScript** – Used for interactive functionality and dynamic elements
* **Google Apps Script** – Used for backend handling of form submissions
* **Google Sheets** – Used as a database for storing booking inquiries

## 👨‍💻 My Role & Contributions

* Implemented responsive layouts to ensure compatibility across devices.
* Created and integrated the booking inquiry form.
* Connected the frontend with Google Sheets using Google Apps Script for automated data storage.

## 📈 Project Outcome

The completed website provides users with an organized view of hotel services and enables customers to submit booking inquiries digitally. The integration with Google Sheets helps maintain customer inquiry records efficiently and reduces manual data management.
